1. Scope
This Privacy Policy applies to the NexRemote Android client application.

2. Product model
The Android app is designed to connect your Android device to a compatible NexRemote Windows host that you own, administer, or are expressly authorized to manage.

The app does not require a Neural Nexus Studios account for its core operation. Neural Nexus Studios does not operate a cloud backend, analytics platform, advertising service, or crash-reporting backend for the normal use of this Android app.

3. Data stored locally on your Android device
Depending on the features you use, the app may store the following data locally:

- app settings such as device name, device identifier, last connected server, connection preferences, haptic settings, gyro settings, and legal acceptance records
- certificate pins or certificate fingerprints used to trust selected NexRemote servers
- locally generated client identity keys stored through Android platform security components
- custom gamepad layouts, macros, and related local control preferences
- local disclosure acknowledgements, including the QR camera disclosure acknowledgement

4. Data accessed or transmitted during use
When you connect to a NexRemote Windows host and use app features, the Android app may access, display, or transmit:

- device names, device identifiers, connection metadata, network endpoints, and certificate fingerprints for discovery, pairing, and trust verification
- remote input commands such as keyboard, mouse, touchpad, touch, joystick, trigger, d-pad, macro, gamepad, and gyro input
- live screen-sharing frames, display metadata, and optional screen-audio stream data
- live camera-streaming frames and camera metadata
- file and folder names, paths, metadata, and file contents used by file-management features
- clipboard content, task-manager information, and media metadata when those features are used

Neural Nexus Studios does not receive this content as part of the app’s normal operation. This data is intended to move only between your Android device and the Windows host you choose to connect.

5. Permissions and sensitive features
The Android app may use the following sensitive capabilities when relevant to the feature you choose:

- camera access only when you choose QR pairing so the app can scan a connection code displayed by the Windows host
- local network and internet access for discovery, pairing, and remote communication
- motion-sensor data such as gyroscope input only when you enable motion-based control features
- vibration and wake behavior for haptics and connection-related device behavior

If you do not grant the required permission, decline the disclosure, or disable the feature, the related feature should remain unavailable or limited.

6. How the app uses data
The Android client uses data only for:

- discovering and connecting to a Windows host selected by the user
- authenticating, pairing, and maintaining trusted connections
- delivering remote-control, streaming, file-management, clipboard, media, task-management, and gamepad features requested by the user
- storing local preferences, legal acknowledgements, trust records, and certificate material
- reliability, troubleshooting, and local device security

The app is not designed to use your data for advertising, profiling, resale, cross-app tracking, or analytics monetization.

7. Sharing and disclosure
Neural Nexus Studios does not sell personal or sensitive user data.

Neural Nexus Studios does not share NexRemote Android app data with advertisers, analytics providers, or unrelated third parties for monetization.

Data may be disclosed only:

- to the Windows host that you choose to connect as part of the app’s core function
- to platform or component providers only to the extent required for normal Android or operating-system functionality
- if required by applicable law, lawful process, or valid governmental request

8. Security
The app supports secure transport and certificate trust validation where available. However, security depends partly on your configuration. If you intentionally use insecure connection modes, disable secure connection preferences, or use an untrusted network, your data may be more vulnerable to interception or misuse.

9. Retention and deletion
Under the current app behavior:

- local preferences, certificate pins, legal acknowledgements, and custom control layouts remain stored locally until you clear app storage or uninstall the app
- Android backup or device-to-device transfer may copy some local app preferences if Android backup is enabled, but excluded secure stores, databases, and log files are not intended to be included under the current configuration
- the app does not currently maintain a developer-hosted account system, so there is generally no separate developer-side account database to delete

You can delete local app data by clearing app storage in Android settings or uninstalling the app.

10. Children
The app is not directed to children under 13. Because the app can expose remote system controls, screens, files, clipboard content, camera content, and process information, it is intended for teens and adults using authorized devices.

11. Changes
Neural Nexus Studios may update this Privacy Policy to reflect product changes, legal requirements, security updates, or store-policy updates. Updated versions may be published in the app, repository, official website, and/or store materials.
